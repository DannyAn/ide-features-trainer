

GENERATE COURSES
To generate specific module you have to:
 # create a directory with a Course name (or it maybe shortname of module)
 # run util.GenerateCourseXml.gen(String courseName, String id, String path)
 # copy result to res/data/

Example Completions:
    courseName: "Completions"
    id: "completions"
    path: "Completions/"

To generate all modules:
 # run util.GenerateCourseXml.main()
 # exclude redundant files from modules.xml list
